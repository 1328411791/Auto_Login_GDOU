# 介绍
广东海洋大学阳江校区校园网登录软件（个人开发），兼容海带主校区等地的校园网，个人纯练手软件

可以解决大部分情况下，web登陆中因为浏览器等原因导致获取acid失败（或错误）的情况

理论上兼容使用深澜协议的校园网，但是没有测试过，如果有兴趣可以试试

本软件仅供学习参考使用
